MAMP Setup Guide for Church Management System

This guide will help you configure the Church Management System to work with your MAMP database.

Step 1: Start MAMP

  1. Open MAMP application
  2. Click "Start Servers" button
  3. Ensure both Apache and MySQL show green lights
  4. Note the MySQL port (usually 8889 for MAMP, 3306 for MAMP Pro)
Important: Make sure MySQL is running before proceeding!

Step 2: Check MAMP Settings

Default MAMP Settings:
MySQL Host: localhost
MySQL Port: 8889 (MAMP) or 3306 (MAMP Pro)
Username: root
Password: root
To check your settings: MAMP → Preferences → Ports

Step 3: Test Database Connection

Click the button below to automatically detect and configure your MAMP database connection:

Test MAMP Connection
This will automatically detect your MAMP settings and update the configuration file.

Step 4: Create Database and Tables

After the connection test passes, create the database structure:

Create Database & Tables

Step 5: Verify Everything Works

Run these tests to ensure everything is working:

Once everything is set up, launch your Church Management System:

Troubleshooting

  • Make sure MAMP is running and MySQL shows a green light
  • Check the MySQL port in MAMP preferences (usually 8889)
  • Try restarting MAMP servers
  • Check if another MySQL service is running on port 3306

  • Default MAMP credentials are usually: root/root
  • Check MAMP preferences for custom MySQL credentials
  • Try accessing phpMyAdmin through MAMP to verify credentials
  • Reset MySQL password in MAMP if needed

  • The system will automatically create the 'church_db' database
  • If it fails, create it manually in phpMyAdmin
  • Run the setup.php script to create tables
  • Make sure MySQL user has CREATE DATABASE privileges
Need Help? Start with the "Test MAMP Connection" button above - it will automatically detect and fix most configuration issues!
